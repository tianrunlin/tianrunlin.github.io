如何在GAE下搭建B3log

  • Xappengine
1、下载并安装java，成功标志：在CMD里面输入 java -version 显示java版本信息则成功，如下图

  附Java SE官方下载地址

2、下载 Google App Engine SDK for Java，解压在根目录下,这里我选C盘，C:\appengine-java-sdk-1.7.2.1
3、下载B3log下载，解压到C盘根目录，C:\b3log-solo-gae-0.5.0
     
4、本地测试，打开CMD，输入cd C:\appengine-java-sdk-1.7.2.1\bin;如图

  输入：dev_appserver.cmd C:\b3log-solo-gae-0.5.0；启动完成后，如图：



  在浏览器中访问：http://localhost:8080，会出现启动画面，表示成功


5、上传至GAE服务器，打开C:\b3log-solo-gae-0.5.0\WEB-INF\appengine-web.xml ，将<application>solo-demo</application>更改为<application>你自己应用的名称</application>,
配置WEB-INF/classes/latke.properties 这个文件的中的 Server 区域，配置成自己的访问地址。（如果要本地测试则改成localhost 端口改为8080）
serverScheme=httpserverHost=你自己的域名serverPort=端口填 80staticServerScheme=httpstaticServerHost=你自己的域名staticServerPort=端口填 80
如图：

然后在CMD里面输入appcfg.cmd update C:\b3log-solo-gae-0.5.0，输入你的GAE账号和密码（输密码的时候不会有显示），

上传完成会出现下图，在浏览器中访问：http://your-ID.appspot.com/init.do 进行初始化。

6、初始化B3log，如图

  点击初始化，弹出对话框，确定后等3秒左右，自动跳转到管理后台，你就可以进入B3log的精彩之旅了……
至此，我的gae下的B3log:http://seuforest.appspot.com/搭建完成
